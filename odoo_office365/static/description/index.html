Setting Rights for Users to use Office365 functionality

  • Once you installed office365, if you can't find office settings in odoo main menu then Enable Developer Mode.
  • Go to Settings --> groups --> Open any Group for which need to add Rights.
  • Add Office365 setting and user setting Access Rights in selected Group.


office 365 Credentials




  • You must have a microsoft account to use this connector.
  • Go to LINK and create an app
  • Click Register App and give Name.
  • Choose the Account Types.



  • After save the apps, you can see your application(client) Id and redirect url.



  • Now add redirect URL for your App. Redirect URL must be http://your-base-url/odoo (this will be used for authentication) and now click on register, as shown below.. The redirect url is used to get authenticating your account and generate token to use your office365 account in Odoo. You have to add /odoo at the end of your redirect url. In my case it is http://localhost:8069/odoo .
  • If you will not add /odoo at the end of your redirect url then you can't use office365 module



  • To get Secret key, Go to Certificate & secrets and click to New Client Secret.



  • After compelete the proccess, copy(Application id,redirect url and secret key).
  • Open Odoo. Go to settings --> users & companies --> office 365.
  • Enter client id and secret(password that you generate earlier), Redirect URI and click Activate.



  • Now go to login user --> Preferences



  • You will see login Office365 button over there.
  • Click Login Button, it will open office365 login page
  • After successfully login in Office365 account. You will be able to get access of your Office365 Account.


Import or Export Data between Office365 and Odoo.




  • After Successfully login Office365 account, here you can see the buttons to perform operation by clicking this buttons you will able to import and export data between Office365 and Odoo.
  • You can activate auto scheduler from here.
  • You can apply date range to import data.


Odoo Calendar

  • Check Import Calender and click Start Execution Button..
  • Now Go to Odoo calendar, here you can see your meetings are synced.


Office 365 Mail

  • You can send messages to the customer by your office 365 Mail.
  • Go to Odoo Contact . open a contact. In the bottom you can send messages to the customer. These messages will be sent to the customer's email by your office 365 mail.
  • You can also send attachemts as well.
  • You can perform same for lead, opportunity, quotaion, sale order, invoice and customer.

What is redirect URL? should redirect URL same with web.base.url?

In our module, redirect url uses to get auth code /permission code of your office365 account. You can set your base URL at your azure app portal and use this URL in Odoo office365 setting.
